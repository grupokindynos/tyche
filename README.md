## Building

To run Plutus simply clone de repository:

```
git clone https://github.com/grupokindynos/plutus
```

Build it or Run it:

```
go build && ./plutus
```

```
go run main.go
```

Make sure the port is configured under en enviroment variable `PORT=8080`
